Install the library

Arduino Software

Please note that ShiftRegister__ArduinoLibrary require the Arduino IDE .

First you need to clone the GitHub repository

$ cd /your/destination/folder
$ git clone https://github.com/VEBERArnaud/ShiftRegister__ArduinoLibrary.git

Then in the Arduino IDE

From the menu: Sketch > Import Library
Select "/your/destination/folder/ShiftRegister__ArduinoLibrary/"

That's it, ShiftRegister__ArduinoLibrary is now ready to use !
